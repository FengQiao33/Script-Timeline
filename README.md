# 直播间脚本时间轴（开场–互动–转化–收尾）

本项目是一个纯前端静态工具，用于**生成直播间脚本时间轴**：将直播流程拆为「开场 / 互动 / 转化 / 收尾」四段，并自动生成每段的脚本卡（目标、主话术、互动设计、转化动作、段尾提醒），同时提供**下一节点倒计时提示**，方便直播过程中随时对节奏。

> 声明：**本项目由阿里云ESA提供加速、计算和保护**

![ESA Pages](./assets/esa-pages-badge.svg)

## 功能特点

- **时间轴脚本卡**：四段结构可编辑，一键生成卡片并支持逐卡复制
- **绝对时间 + 相对时间**：可设置直播开始时间，自动计算每段的时间点（如 `20:30 · T+22m`）
- **倒计时提示**：按“下一节点”动态倒计时（支持暂停/继续；支持将“当前时间”设为开播时间）
- **TXT 导出**：一键下载脚本时间轴 TXT，可直接发给主播/运营同学
- **零依赖静态站点**：无需构建，直接部署

## 使用方式

1. 打开网站后，填写：
   - 直播开始时间（可选）
   - 总时长（分钟）
   - 四段脚本内容与时长
2. 点击 **生成脚本卡**
3. 需要文件时，点击 **下载脚本 TXT**

## 部署到阿里云 ESA Pages

这是一个静态站点项目。部署时：

- 直接将仓库内容上传/关联到 Pages
- 入口文件为：`index.html`

## 项目介绍（给评委）

- **实用性**：直播间节奏把控往往依赖经验与临场发挥，本工具把“开场–互动–转化–收尾”标准流程固化为可编辑模板，减少运营准备时间，提升话术一致性与转化动作执行率。
- **创意性**：用“下一节点倒计时 + 脚本卡”把直播控场过程可视化，主播无需翻文档，只看屏幕即可对节奏；并支持一键把“现场时间”校准为开播时间，适应临场延迟。
- **技术深度**：在纯静态前端中实现了结构化脚本编辑、时间轴计算、倒计时调度、复制与文件导出，并通过本地存储保持配置，保证“部署即用、打开即用”。
